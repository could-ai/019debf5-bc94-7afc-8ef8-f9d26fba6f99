# Minecraft AI Tvořič

Tato aplikace slouží jako AI asistent a vývojářské prostředí pro generování obsahu do hry Minecraft. Můžete s pomocí AI chatu vytvářet vlastní mody, pluginy, datapacky nebo dokonce generovat unikátní vlastní biomy.

## Klíčové vlastnosti

- **AI Chatový Asistent:** Komunikujte s AI pro generování specifického obsahu do Minecraftu.
- **Výběr Typu Projektu:** V horním menu si můžete vybrat, zda pracujete na Modu, Pluginu, Datapacku nebo Vlastním biomu.
- **Podpora Souborů:** Možnost nahrávat existující skripty, modely nebo textury z počítače/mobilu přímo do chatu pomocí tlačítka `+`.
- **Moderní Tmavý Design:** Tmavé téma laděné do zelených barev (Dark Mode) šetrné k očím při delší práci.

## Použité technologie

- **Frontend:** Flutter & Dart (Multiplatformní podpora - iOS, Android, Windows, macOS, Linux, Web)
- **UI/UX:** Material 3 Design

## Jak spustit projekt

1. Ujistěte se, že máte nainstalovaný Flutter SDK.
2. Naklonujte repozitář nebo stáhněte zdrojové kódy.
3. Spusťte příkaz pro instalaci závislostí:
   ```bash
   flutter pub get
   ```
4. Spusťte aplikaci na vybraném zařízení:
   ```bash
   flutter run
   ```
